# VixenOS

A UGC (User Generated Content) platform built with TypeScript and Next.js. Designed for content creators to publish, manage, and distribute their work.

## Features

- Content publishing and management
- User profiles and authentication
- Feed and discovery system
- Responsive, mobile-first UI

## Tech Stack

- Next.js (App Router)
- TypeScript
- CSS Modules

## Running Locally

```
npm install
npm run dev
```
